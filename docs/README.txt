Ecologix Solutions - Project Package

Structure:
- index.html
- about.html
- projects.html
- gallery.html
- team.html
- contact.html
- css/style.css
- js/scripts.js
- js/lang.json
- js/config.json
- img/ (add your images here)
  - Logo2.png (header logo)
  - Logo3.png (footer logo, monochrome)
- videos/ (add mp4 files here)
- docs/README.txt (this file)

How to run locally:
- Use Live Server in VS Code, or run:
  python -m http.server 8080
  and open http://localhost:8080

Config:
- Edit js/config.json to set defaultLang, enabledLangs, formEndpoint, corporateEmail
- Edit js/lang.json to change translations
